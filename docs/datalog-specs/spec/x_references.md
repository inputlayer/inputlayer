# References

- [Foundations of Databases](http://webdam.inria.fr/Alice/) - Abiteboul, Hull, Vianu. Comprehensive textbook on database theory including Datalog.

- [What You Always Wanted to Know About Datalog](https://dl.acm.org/doi/10.1145/588111.588141) - Classic survey paper on Datalog.

## InputLayer

- [InputLayer Repository](https://github.com/inputlayer/inputlayer) - Source code and documentation.

- [Differential Dataflow](https://github.com/TimelyDataflow/differential-dataflow) - The computation engine powering InputLayer.

- [Unicode Standard](http://www.unicode.org/versions/latest/) - Character encoding.

- [IEEE 754](https://standards.ieee.org/ieee/754/993/) - Floating-point arithmetic.
